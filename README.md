# Hockey-T64:
This is a simple Pong inspired game, themed after hockey tables, created using Pygame. <br />
## Use Case:  <br />
![Hockey-T64 USECASE](useCase.png) <br />
## Installation
1. Clone this repository
2. Install the required dependency pygame
3. Run the main.py file
## Usage
Z, S for PLAYER1 movement <br />
ARROWUP, ARROWDOWN for PLAYER2 movement
